# 赞助 ECC

ECC 作为开源智能体 Harness 性能系统，在 Claude Code、Cursor、OpenCode 和 Codex 应用/CLI 上运行。

## 为什么赞助

赞助直接资助：

- 更快的错误修复和发布周期
- 跨 Harness 的跨平台一致性工作
- 对社区免费的公共文档、技能和可靠性工具

## 赞助层级

这些是实用的起点，可根据合作范围调整。

| 层级 | 价格 | 最适合 | 包含内容 |
|------|-------|----------|----------|
| 试点合作伙伴 | $200/月 | 首次赞助合作 | 每月指标更新、路线图预览、优先维护者反馈 |
| 成长合作伙伴 | $500/月 | 积极采用 ECC 的团队 | 试点权益 + 每月办公时间同步 + 工作流集成指导 |
| 战略合作伙伴 | $1,000+/月 | 平台/生态系统合作 | 成长权益 + 协调发布支持 + 更深入的维护者合作 |

## 赞助报告

每月共享的指标可包括：

- npm 下载数（`ecc-universal`、`ecc-agentshield`）
- 仓库采用情况（星标、分叉、贡献者）
- GitHub App 安装趋势
- 发布节奏和可靠性里程碑

有关确切的命令片段和可重复的拉取流程，请参阅 [`docs/business/metrics-and-sponsorship.md`](docs/business/metrics-and-sponsorship.md)。

## 期望与范围

- 赞助支持维护和加速；不转移项目所有权。
- 功能请求根据赞助层级、生态系统影响和维护风险进行优先排序。
- 安全和可靠性修复优先于全新功能。
