# 质量门控命令

按需运行 ECC 质量管线，用于文件或项目范围。

## 用法

`/quality-gate [path|.] [--fix] [--strict]`

- 默认目标：当前目录（`.`）
- `--fix`：允许在已配置的地方自动格式化/修复
- `--strict`：在支持的地方将警告视为失败

## 管线

1. 检测目标的语言/工具。
2. 运行格式化器检查。
3. 在可用时运行 lint/类型检查。
4. 生成简洁的修复列表。

## 说明

此命令反映钩子行为，但由操作员主动调用。

## 参数

$ARGUMENTS:
- `[path|.]` 可选目标路径
- `--fix` 可选
- `--strict` 可选
