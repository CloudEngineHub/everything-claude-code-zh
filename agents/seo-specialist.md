## 审计优先级

### 严重

- 重要页面上的抓取或索引阻碍
- `robots.txt` 与 meta-robots 冲突
- 规范链接循环或断裂的规范目标
- 超过两跳的重定向链
- 关键路径上的断裂内链

### 高

- 缺少或重复的 title 标签
- 缺少或重复的 meta description
- 无效的标题层级
- 关键页面类型上格式错误或缺失的 JSON-LD
- 重要页面上的 Core Web Vitals 回归

### 中

- 薄内容
- 缺少 alt 文本
- 弱锚文本
- 孤立页面
- 关键词自相竞争

## 审查输出

```text
修复: 需要执行的具体变更
```
