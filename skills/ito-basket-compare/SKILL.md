# Itô 篮子比较

使用此技能将篮子、主题或市场集与用户的知识库、投资组合笔记、研究备忘录、CRM 上下文或陈述的论文进行比较。

此技能是只读的。它不推荐交易。它帮助用户在决定做什么之前检查契合度、敞口、假设和缺失上下文。

## 防护

- 不提供投资建议或告诉用户买入、卖出、持有、对冲、杠杆或调整交易规模。
- 不执行、准备或提交订单。
- 除非用户明确指向，否则不使用私有文档。
- 仅在明确的用户请求后对只读 Itô 篮子/市场数据使用 `ITO_API_KEY`。
- 如果与财务比较，请保护隐私并仅汇总比较所需的字段。

## 比较模式

### 篮子与知识库

1. 识别篮子主题和标的。
2. 检索用户的相关笔记、文档或记忆片段。
3. 将每个标的映射到声明、来源、不确定性和陈旧假设。
4. 返回对齐的信号、冲突的信号和缺失的研究。

### 篮子与投资组合笔记

1. 解析用户的观察名单、持股摘要或敞口笔记。
2. 比较主题、地理位置、时间范围和事件结果。
3. 标记集中度、相关性和重复的叙事敞口。
4. 避免建议；将输出表述为检查和问题。

### 篮子与财务上下文

1. 仅接受用户提供或明确选择的财务上下文。
2. 识别流动性、回撤、时间范围和约束不匹配。
3. 询问缺失的约束而非猜测。

## 输出合约

使用此结构：

1. 篮子摘要
2. 比较目标
3. 匹配
4. 冲突或陈旧假设
5. 缺失上下文
6. 用户操作检查清单

```text
此比较仅供参考，并非投资或交易建议。
```
