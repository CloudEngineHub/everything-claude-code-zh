# Itô 交易规划器

当用户想要一个用于预测市场想法、篮子调整、平台比较或手动执行计划的结构化工作表时使用此技能。

此技能有意设计为非执行性的。它产生用户可以手动审查的检查清单和参数表。

## 防护措施

- 不要说某个交易是好的、坏的、最优的或推荐的。
- 不要提供投资建议或仓位大小建议。
- 不要执行、取消、路由或签署订单。
- 不要请求私钥、助记词、交易所密码或钱包凭证。
- 在任何工作流从研究转向可执行工具之前，要求用户明确批准。

## 规划工作流

1. 将用户的想法复述为中性假设。
2. 识别市场、平台、标的、解析规则、费用和数据新鲜度约束。
3. 如果已配置 `ITO_API_KEY` 且用户要求，读取 Itô 篮子元数据。
4. 构建手动工作表：
   - 市场/标的
   - 平台
   - 数据源
   - 当前可观察价格或状态
   - 解析规则
   - 流动性注意事项
   - 未解决问题
   - 手动操作链接或下一步审查步骤
5. 在讨论自动化、密钥、平台认证或资金约束之前运行 `prediction-market-risk-review`。

## 允许的语言

使用：

- "手动规划工作表"
- "行动前需要回答的问题"
- "可观察的平台数据"
- "风险和约束审查"

避免：

- "你应该买入/卖出"
- "最佳交易"
- "保证"
- "无风险"
- "最优规模"

## 输出契约

每个计划以以下文本结束：

```text
这是一份规划工作表，不是投资或交易建议。请自行审查平台规则并做出交易决策。
```
