# 企业智能体运维

此技能用于需要超越单一 CLI 会话的运维控制的云托管或持续运行的智能体系统。

## 运维领域

1. 运行时生命周期（启动、暂停、停止、重启）
2. 可观测性（日志、指标、追踪）
3. 安全控制（范围、权限、终止开关）
4. 变更管理（上线、回滚、审计）

## 基线控制

- 不可变的部署产物
- 最小权限凭证
- 环境级别的密钥注入
- 硬超时和重试预算
- 高风险操作的审计日志

## 跟踪指标

- 成功率
- 每个任务的平均重试次数
- 恢复时间
- 每个成功任务的成本
- 失败类别分布

## 事件模式

当故障激增时：
1. 冻结新上线
2. 捕获代表性追踪
3. 隔离失败路由
4. 使用最小的安全更改进行修补
5. 运行回归 + 安全检查
6. 逐步恢复

## 部署集成

此技能配合使用：
- PM2 工作流
- systemd 服务
- 容器编排器
- CI/CD 门控
